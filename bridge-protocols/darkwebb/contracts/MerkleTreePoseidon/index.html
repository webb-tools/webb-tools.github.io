    MerkleTreePoseidon.sol¶

    Implementation

    Description¶

    The MerkleTreePoseidon is an deployment ready implementation of a MerkleTreeWithHistoryPoseidon.sol and inherits all its methods. It populates the zero hashes and implements the hash function for contracts that inherit it.
